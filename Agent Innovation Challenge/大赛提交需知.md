### 大赛提交需知

#### 1. 参考样例

参赛队伍的提交形式可以参考路径参考样例下的文件结构，需要包含文件

- Agent Tool场景及技术方案介绍，可以用英文或中文

- Agent Tool代码，需包含必要的README，保证代码的可运行性

- Agent Tool Demo，**Tool需要提前部署服务，并在openJiuwen Studio中添加集成，Demo基于OpenJiuwen Studio的界面来演示**

以上三部分内容必须包含，但代码结构、文档格式仅供参考并无限制

#### 2. 提交格式
参赛队伍应当在【比赛提交】文件夹下提交
命名方式采用：队伍名称+Tool名称
