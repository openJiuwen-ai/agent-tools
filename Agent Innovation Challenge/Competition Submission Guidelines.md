### Competition Submission Guidelines
#### 1. Reference Example
Teams can refer to the file structure under the "Reference Example" directory for submission format. The submission must include the following files:
- Introduction to the Agent Tool scenario and technical solution, which can be written in English or Chinese.

- Agent Tool code, including a necessary README file to ensure the code is runnable.

- Agent Tool Demo, **where the tool needs to be pre-deployed and integrated into OpenJiuwen Studio. The demo should demonstrate the tool based on the interface of OpenJiuwen Studio.**

The above three parts are mandatory, but the code structure and document format are for reference only and are not restricted.
#### 2. Submission Format
Teams should submit their entries in the [Competition Submission] folder.
The naming convention should be: Team Name + Tool Name
