# 查天气Tool

## 1. Tool应用场景

### 1.1 核心场景定义

查天气Tool是一个**专用、可复用的工具组件**，允许Agent获取实时天气信息、未来预报和历史天气数据，帮助用户做出基于天气的决策。

### 1.2 具体应用场景

| 场景类别         | 典型用户需求                         | 决策价值                     |
| :--------------- | :----------------------------------- | :--------------------------- |
| **日常生活**     | 明日是否需带伞？周末适合户外活动吗？ | 出行便利性、活动安排优化     |
| **差旅规划**     | 出差地气温如何？需要准备什么衣物？   | 行李准备恰当性、行程舒适度   |
| **专业领域**     | 农业种植浇水决策；物流运输路线规划   | 降低风险、提升效率、减少损失 |
| **智能家居联动** | 根据天气自动开关窗户、调节空调       | 能源节约、居家自动化         |
| **健康关怀**     | 敏感人群（如哮喘患者）的空气质量提醒 | 健康预警与防护               |

### 2.1 工具定义（标准化接口）

```
{
  "tool_name": "weather_query",
  "description": "查询实时天气、多日预报及历史天气数据",
  "version": "1.2.0",
  "parameters": {
    "location": {
      "type": "string",
      "description": "查询地点（支持城市名、邮政编码、坐标等格式）",
      "required": true
    },
    "query_type": {
      "type": "string",
      "description": "查询类型：realtime/forecast/history",
      "required": true,
      "enum": ["realtime", "forecast", "history"]
    },
    "date_range": {
      "type": "object",
      "description": "日期范围（用于forecast和history类型）",
      "required": false,
      "properties": {
        "start_date": {"type": "string", "format": "YYYY-MM-DD"},
        "end_date": {"type": "string", "format": "YYYY-MM-DD"},
        "days": {"type": "integer", "minimum": 1, "maximum": 14}
      }
    },
    "units": {
      "type": "string",
      "description": "单位制：metric/imperial",
      "required": false,
      "default": "metric"
    },
    "language": {
      "type": "string",
      "description": "返回语言代码",
      "required": false,
      "default": "zh-CN"
    }
  },
  "returns": {
    "success": {"type": "boolean"},
    "data": {"type": "object"},
    "error": {"type": "string"},
    "timestamp": {"type": "string"}
  }
}
```
